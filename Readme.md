# Simple Stock Price application
 ## steps to run
 1. Install Streamlit 
 ```
 pip install streamlit
 ```
 2. Install yfinance
 ```
 pip install yfinance
 ```
 3. Open Command prompt
 ```
 streamlit run Firstapp.py
 ```
